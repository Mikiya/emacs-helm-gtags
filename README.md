helm-gtags.el
==================

Introduction
------------
`helm-gtags.el` is front-end of `gtags.el` with `helm`

**helm-gtags.el** is not compatible **anything-gtags.el**

*WARNINGS: THIS IS ALPHA VERSION.*

Requirements
------------
* Emacs 22.1 or higher.
* GNU Global 5.7 or higher(`global` and `gtags.el`)
